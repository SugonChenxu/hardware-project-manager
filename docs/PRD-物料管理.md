# PRD - 物料管理

> 版本: v1.4 | 日期: 2026-07-01 | 状态: 已实现

---

## 1. 模块概述

硬件项目的物料清单管理。以 **Excel 导入为主**，手动添加为辅。所有字段均可像 Excel 一样点击内联编辑。

## 2. 数据模型

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| seq | number | 是 | 序号，自动递增 |
| materialStatus | MaterialStatus | 是 | 物料状态（5档预设），默认「默认」 |
| manufacturer | string | - | 厂家/品牌 |
| model | string | - | 物料型号/规格 |
| partNumber | string | - | 物料号/料号 |
| quantityPerSet | number | - | 单套用量 |
| setCount | number | - | 套数 |
| totalQuantity | number | 计算 | 总数量 = 单套用量 × 套数（前端自动计算） |
| purchaseDate | string | - | 采购时间 |
| leadTime | number | - | 采购周期（天） |
| expectedDelivery | string | 计算 | 预计交期 = 采购时间 + 采购周期（前端自动计算） |
| notes | string | - | 补充备注 |

### MaterialStatus 枚举

| 状态 | 颜色 | 背景色 | 说明 |
|------|------|--------|------|
| 默认 | #8c8c8c | #f5f5f5 (灰) | 新物料初始状态 |
| 已入库 | #fff | #52c41a (绿) | 物料已到货入库 |
| 已下单 | #135200 | #b7eb8f (浅绿) | 已下采购单 |
| 待决策 | #fff | #faad14 (黄) | 需要决策/评审 |
| 高风险 | #fff | #ff4d4f (红) | 延期/断供等风险 |

### 自动状态切换规则

- 物料初始状态 = **默认**
- 当用户设置「采购时间」时，如果当前状态为「默认」，**自动切换为「已下单」**
- Excel 导入时同样触发此规则
- 批量修改采购时间时同样触发此规则

## 3. 列顺序

`序号 → 物料状态 → 厂家 → 物料型号 → 物料号 → 单套用量 → 套数 → 总数量 → 采购时间 → 采购周期 → 预计交期 → 备注`

## 4. 功能清单

### 4.1 Excel 导入

- 点击「导入Excel」按钮，选择 .xlsx/.xls 文件
- 自动识别列名（支持中英文常见列名映射）
- 弹窗预览前 50 行数据
- 确认后批量写入，自动分配序号

**支持的列名映射：**

| Excel列名 | 映射字段 |
|-----------|---------|
| 序号/seq/no | seq |
| 物料状态/状态/status | materialStatus |
| 厂家/供应商/品牌/manufacturer | manufacturer |
| 物料型号/型号/规格/model | model |
| 物料号/料号/编号/partNumber | partNumber |
| 单套用量/单套数量 | quantityPerSet |
| 套数/总套数 | setCount |
| 采购时间/采购日期 | purchaseDate |
| 采购周期/周期 | leadTime |
| 备注/说明 | notes |

### 4.2 手动添加

- 点击「添加一行」直接在表格末尾插入新行
- 新行 materialStatus 默认 = 「默认」
- 插入后自动聚焦物料号列，可立即输入

### 4.3 物料状态列

- 位于序号和厂家之间，列宽 80px，居中
- 渲染为紧凑**彩色标签**，带 ▾ 小箭头提示可交互
- **点击标签**弹出 Dropdown 菜单，5 个选项均带彩色预览
- 选中后标签立即更新，完美适配列宽

### 4.4 全表格自由内联编辑

- 所有可编辑单元格点击即进入编辑模式
- 编辑态单元格高亮，失焦/回车保存，Esc 取消
- Tab 键横向跳转下一个可编辑单元格，Shift+Tab 回退
- 文本字段：直接输入
- 数字字段：InputNumber
- 采购时间：DatePicker 组件

### 4.5 行选择 + 批量操作

- 表格第一列为 checkbox 多选，支持全选/反选/清除
- 选中行后顶部蓝色批量操作栏显示：

**批量修改所有字段流程：**
1. 勾选目标物料行
2. 在蓝色栏中点击「选择批量修改的字段」下拉，可选：厂家、物料型号、物料号、单套用量、套数、采购时间、采购周期、物料状态、备注
3. 选择字段后，右侧出现对应类型的输入控件：
   - 文本字段（厂家/型号/料号/备注）→ Input 文本框
   - 数字字段（单套用量/套数/采购周期）→ InputNumber
   - 日期字段（采购时间）→ DatePicker
   - 状态字段（物料状态）→ 彩色 Select 下拉
4. 输入值后点击「应用」或按回车，批量更新所有选中行
5. 弹出成功提示，输入自动清空

- **「批量删除」按钮** — 二次确认后一次性删除

### 4.6 排序

- 按厂家（中文拼音）排序，默认降序，点击切换升/降序
- ✕ 取消排序恢复默认序号排列

### 4.7 撤销导入

- 每次批量导入后，工具栏显示「撤销导入」按钮
- 点击后移除最近一次导入的全部物料，数据恢复导入前状态
- 撤销按钮仅在最近一次导入的 5 分钟内可见（页面刷新后仍有效）
- 二次导入会覆盖上一次的撤销快照，只能撤销最近一次

### 4.8 导出 Excel

- 导出当前项目全部物料清单为 .xlsx
- 包含物料状态、总数量、预计交期等列

### 4.9 搜索过滤

- 支持料号、厂家、型号、物料状态、备注模糊搜索

## 5. 交互细节

| 场景 | 行为 |
|------|------|
| 设置采购时间 + 状态=默认 | 自动切换为「已下单」 |
| 批量设采购时间 | 默认状态物料自动切「已下单」 |
| 预计交期已过期 | 日期变红 + ⚠️ 图标 |
| 无采购时间或无周期 | 预计交期显示 `-` |
| 点击单元格 | 进入内联编辑，自动聚焦 |
| Tab 键 | 跳到下一个可编辑单元格 |
| 选中行后 | 顶部出现蓝色批量操作栏 |
| 选择批量修改字段 | 右侧出现对应输入控件 |
| 输入值后回车 | 等效于点击「应用」按钮 |
| 取消选择 | 所有选中行清空，批量栏消失 |
| 导入确认后 | 工具栏出现「撤销导入」按钮，5分钟内有效 |
| 撤销导入 | 移除最近一次导入的全部物料，数据恢复 |
| 再次导入 | 覆盖上一次撤销快照，只能撤销最近一次 |

## 6. 技术实现

| 项目 | 方案 |
|------|------|
| Excel 解析 | xlsx (SheetJS) |
| 状态管理 | Zustand (useMaterialStore) |
| 数据持久化 | localStorage |
| 日期处理 | dayjs |
| 行选择 | Ant Design Table rowSelection |
| 批量操作 | useMaterialStore.batchRemove / batchUpdate / undoImport |
