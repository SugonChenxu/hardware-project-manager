# PRD — 每日待办

> 版本: v1.0 | 日期: 2026-07-01 | 状态: 已实现

---

## 1. 模块概述

**模块名称**：每日待办  
**页面路由**：`/tasks`  
**目标用户**：硬件项目管理者 X（个人使用）  
**核心定位**：快速记录和跟踪每日任务，支持优先级分类、子任务拆解、时间和责任人标记。

**设计原则**：
- 极简录入：无需弹窗，回车即添加，光标不离开输入框
- 一眼看完：待办 / 已完成 双栏布局
- 按重要程度驱动颜色：红色 = 紧急/高优先，黑色 = 中/低优先

## 3. 功能清单

### 3.1 快速录入（待办列顶部）

| 项目 | 规格 |
|------|------|
| 触发方式 | 永远可见的输入框，placeholder: "输入任务，回车添加..." |
| 添加行为 | 回车 → 创建任务（默认优先级=中），清空输入框，光标保留 |
| 空值处理 | 空内容回车不创建 |

### 3.2 任务行（待办列）

每条任务从左到右包含：

| 序号 | 元素 | 说明 |
|------|------|------|
| 1 | 状态灯 | 圆形，颜色对应优先级；点击弹出下拉菜单选择 |
| 2 | 展开箭头 | ▶ 灰色/蓝色列表图标；点击展开子任务面板 |
| 3 | 任务标题 | inline input，回车失焦并跳回快速录入框；失焦自动保存（空则删除） |
| 4 | 截止日期 | 点击弹出 DatePicker，选中后显示 `M/D` 格式 |
| 5 | 责任人 | inline input，直接输入名字 |
| 6 | 完成按钮 | 绿色边框方框，hover 填充绿色，点击移入已完成列 |
| 7 | 删除按钮 | hover 出现，带二次确认 |

**文本颜色规则：**
- 紧急 / 高 → `#ff4d4f`（红色），紧急加粗
- 中 / 低 → `#333`（黑色）

**优先级状态灯颜色：**

| 优先级 | 颜色 | 色值 |
|--------|------|------|
| 紧急 | 深红 | `#cf1322` |
| 高 | 红色 | `#ff4d4f` |
| 中 | 黄色 | `#faad14` |
| 低 | 绿色 | `#52c41a` |

### 3.3 子任务（展开后）

| 项目 | 规格 |
|------|------|
| 触发 | 点击 ▶ 箭头，展开面板自动聚焦输入框 |
| 添加 | 输入 + 回车，光标保留可连续添加 |
| 勾选 | 点击方框 → 打勾 + 删除线（完成）；再次点击撤销 |
| 删除 | hover 出现删除按钮，带二次确认 |
| 标识 | 有子任务时箭头变蓝色列表图标，tooltip 显示 `(未完成/总数)` |

### 3.4 已完成列

| 项目 | 规格 |
|------|------|
| 任务标题 | 灰色 + 删除线 |
| 状态灯 | 保留原色但 35% 透明度 |
| 日期/责任人 | 只读展示（如有） |
| 完成时间 | `✓ M/D` 格式 |
| 子任务 | 列表图标 + tooltip `已完成数/总数` |
| 撤销 | hover 出现回退箭头，点击回到待办列 |
| 删除 | hover 出现，带二次确认 |

## 4. 数据结构

```typescript
interface SubTask {
  id: string;
  title: string;
  completed: boolean;
}

interface Task {
  id: string;
  projectId: string;
  title: string;
  description: string;
  status: 'todo' | 'in_progress' | 'review' | 'done';
  priority: 'low' | 'medium' | 'high' | 'urgent';
  assignee: string;
  startDate: string;
  endDate: string;
  dependencies: string[];
  textColor: string;
  completedAt: string;
  subTasks: SubTask[];
  createdAt: string;
  updatedAt: string;
}
```

**存储**：`localStorage` key: `tasks`，通过 `useTaskStore` (Zustand) 管理。

---

## 5. 交互细节

| 场景 | 行为 |
|------|------|
| 空标题失焦 | 自动删除该任务 |
| 已删除的任务有子任务 | 一并删除（子任务嵌套在 Task 内） |
| 主任务完成 | 子任务保留（不自动完成） |
| 主任务撤销 | 回到待办，子任务状态不变 |
| 页面刷新 | 数据从 localStorage 恢复 |
